    Mermaidは、Markdownにインスパイアされたテキスト定義からダイアグラムを作成するJavaScriptベースのダイアグラミングおよびチャートツールです。

    flowchart TD
    A[開始] --> B{動作していますか?}
    B -->|はい| C[素晴らしい!]
    B -->|いいえ| D[デバッグ]
    D --> B
    C --> E[終了]

    ガントチャート¶

    gantt
    title プロジェクトスケジュール
    dateFormat  YYYY-MM-DD
    section 計画
    要件収集                 :a1, 2023-01-01, 10d
    設計                    :a2, after a1, 15d
    section 開発
    実装                    :a3, after a2, 20d
    テスト                  :a4, after a3, 10d
    section デプロイ
    デプロイ                :a5, after a4, 5d

    ドキュメントでのダイアグラムの使用¶

    ドキュメントにダイアグラムを含めるには：

    1. PlantUMLの場合、次の構文を使用します： 

      ```plantuml
@startuml
// ここにPlantUMLコードを記述
@enduml
```

    2. Mermaidの場合、次の構文を使用します： 

      ```mermaid
// ここにMermaidコードを記述
```

    両方のダイアグラムタイプは、ドキュメント内で自動的にレンダリングされます。
